# Mies

The **Tanqory Mies design system** is the foundation for building elegant, minimal, and consistent Tanqory web- and mobile-app experiences.

## What's the Tanqory admin?
Merchants run their entire commerce operation inside the **Tanqory admin**.  
Here they set up stores, configure settings, and manage daily business—products, orders, customers, payments—everything in one place.

## What is Tanqory Mies?
**Tanqory Mies** is the shared design language for the Tanqory platform.  
It provides design guidance, code libraries, developer standards, and API documentation so teams can create high-quality merchant experiences that feel seamless across web and mobile.

## Who is Mies for?
Mies is for everyone building on Tanqory:

- **Internal teams** creating first-party features  
- **External designers and developers** extending Tanqory with custom apps or integrations

## What are Tanqory apps?
The admin supports installable apps distributed through the **Tanqory App Store**.  
Apps let merchants add new capabilities without leaving the familiar admin interface.  
By building with **Mies** and Tanqory's App SDKs and following the **App Design Guidelines**, developers can deliver a unified, streamlined experience that matches the rest of the Tanqory environment.
