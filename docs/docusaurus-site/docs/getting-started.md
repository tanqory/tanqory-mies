# Getting Started

**Less is more, enforced by design.**

Tanqory Mies is the cross-platform design system that enforces clarity, minimalism, and precision for global commerce interfaces on Web and Mobile.

## Quick Start

Choose your platform and get productive in minutes:

### Web Development
```bash
npm install @tanqory/mies-core-web @tanqory/mies-tailwind
```

### Mobile Development  
```bash
npm install @tanqory/mies-core-native @tanqory/mies-tokens
```

### Design System Enforcement
```bash
npm install --save-dev eslint-plugin-tanqory-mies
```

## UI reduced to its essence

Every pixel serves a purpose. Every component drives conversion. Every constraint unlocks creativity.

Built for the future of commerce, Mies eliminates visual noise to spotlight what matters most: your products, your customers, your success.

## Systematic Constraints

- **Three colors only**: Brand lime (#E1FF00), pure black, pure white
- **Sharp edges**: No rounded corners, no exceptions
- **Flat surfaces**: Zero shadows, zero gradients, zero depth effects
- **Precise spacing**: 4, 8, 12, 16, 24, 32 pixel scale — nothing else

## Cross-Platform Precision

One design system. Two platforms. Zero compromises.

The same tokens, the same components, the same uncompromising quality across Web and React Native. Your brand stays consistent whether customers shop on desktop, mobile web, or native apps.

## Performance First

- **Zero-JS by default** with React Server Components
- **Tree-shakable imports** — use only what you need
- **Sub-1KB components** — every byte optimized
- **CSS variables** — no runtime overhead
